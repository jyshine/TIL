Java의 java.util.Calendar 클래스는 날짜와 시간 값을 조작하는 데 사용됩니다.
이 클래스는 연도, 월, 일, 시, 분, 초와 같은 날짜 및 시간 구성 요소를 검색하고 조작하는 일련의 메서드를 제공합니다.

Calendar는 추상 클래스이므로 직접 인스턴스를 만들 수 없습니다. 대신에 GregorianCalendar와 같은 하위 클래스를 사용해야 합니다.
GregorianCalendar는 가장 일반적으로 사용되는 Calendar의 구현입니다.

Calendar 클래스를 사용하여 수행할 수 있는 일반적인 작업은 다음과 같습니다.

- 현재 날짜와 시간 가져오기
- 특정 날짜와 시간 설정
- 연도, 월, 일, 시, 분, 초와 같은 특정 구성 요소의 값을 검색하기
- 날짜 또는 시간에서 지정된 시간 양을 추가하거나 뺄 수 있는 기능
- 특정 방식으로 날짜 또는 시간 서식 지정하기
- 전반적으로, java.util.Calendar 클래스는 자바에서 날짜 및 시간을 다루는 데 편리한 방법을 제공하며, 날짜 및 시간 계산 또는 서식 지정이 필요한 애플리케이션에서 자주 사용됩니다.
