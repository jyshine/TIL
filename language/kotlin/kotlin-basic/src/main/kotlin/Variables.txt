var (Mutable 변수):
var로 선언된 변수는 가변(mutable) 변수입니다. 이 변수의 값은 변경될 수 있습니다.
var로 선언된 변수는 재할당이 가능하므로 동일한 변수 이름으로 다른 값을 할당할 수 있습니다.

var x = 10
x = 20 // 가능: x 변수의 값을 변경할 수 있음

val (Immutable 변수):
val로 선언된 변수는 불변(immutable) 변수입니다. 이 변수의 값은 한 번 할당되면 변경될 수 없습니다.
val로 선언된 변수는 재할당이 불가능하므로 한 번 값을 할당하면 나중에 값을 변경할 수 없습니다.

val y = 30
// y = 40 // 불가능: val 변수는 재할당할 수 없음

간단히 말해서,
var는 값이 변경될 수 있는 변수를 나타내고,
val은 값이 변경될 수 없는(read-only) 변수를 나타냅니다.

불변 변수(val)를 사용하면 프로그램의 안정성을 높이고 버그를 방지하는 데 도움이 됩니다.
때문에 변수의 값을 변경할 필요가 없는 경우에는 val을 사용하는 것이 권장됩니다.
