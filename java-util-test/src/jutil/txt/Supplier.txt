Java에서 Supplier는 매개 변수가 없는 메서드를 나타내는 함수형 인터페이스입니다.
즉, get() 메서드 하나를 갖습니다.

Supplier 인터페이스는 다른 함수형 인터페이스와 마찬가지로 람다 표현식과 메서드 참조를 사용하여 인스턴스화 할 수 있습니다.
주로 객체를 생성하거나, 데이터를 가져오거나, 계산을 지연하는 데 사용됩니다.





          +--------------+
          |   Supplier   |
          +--------------+
          | + get()      |
          +--------------+
                   ^
                   |
          +-----------------+
          |                 |
      +------+          +--------+
      | ...  |          | LongSupplier |
      +------+          +--------+
                   ^
                   |
           +-----------------+
           |                 |
       +------+          +------------+
       | ...  |          | IntSupplier |
       +------+          +------------+
                   ^
                   |
           +-----------------+
           |                 |
      +------+          +---------------+
      | ...  |          | DoubleSupplier|
      +------+          +---------------+
                   ^
                   |
          +-----------------+
          |                 |
      +------+          +----------------+
      | ...  |          | BooleanSupplier|
      +------+          +----------------+
